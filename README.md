# Jornada de Estudos em Java 
Bem-vindo ao meu repositório de estudos em Java! 

O objetivo deste espaço é documentar minha evolução na linguagem Java, partindo de conceitos fundamentais até tópicos mais avançados.

## O que estou aprendendo
Este repositório funciona como um diário de bordo técnico. Meu foco é entender profundamente a lógica da linguagem, estruturas de dados, Orientação a Objetos e boas práticas de desenvolvimento.

## Aprendizado em Dobro
Além da prática com Java, este repositório é o campo de treinamento para **aperfeiçoar o uso do Git e GitHub**. Estou em fase de aprendizado sobre versionamento de código, gestão de histórico e fluxo de trabalho com repositórios remotos. Cada *commit* aqui é uma etapa na construção do meu portfólio profissional.

## Estrutura de Organização
Os exercícios estão organizados por nível de complexidade:
- **Básico:** Sintaxe, variáveis e operadores.
- **Intermediário:** Estruturas de repetição e lógica.
- **Avançado:** POO, coleções e padrões de projeto.

## Tecnologias
- **Linguagem:** Java
- **IDE:** IntelliJ IDEA
- **Controle de Versão:** Git & GitHub
